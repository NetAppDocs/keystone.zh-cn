    创建磁盘

    04/18/2022

    本节介绍如何通过直接指定磁盘详细信息来创建新磁盘。有关如何基于现有磁盘的 Snapshot 创建磁盘的说明，请参见 "从 Snapshot 创建磁盘"。

    开始之前

    • 如果块存储可用但未初始化，则系统会要求您先对其进行初始化，然后才能创建磁盘。要初始化块存储，您需要：

      • 块存储的子网。如果您不熟悉网络环境，请咨询 IT 部门以了解相关值。

      • 要使用的协议。默认情况下，块存储已启用 iSCSI 服务。确保已创建子网，以便为指定区域中的子租户初始化块存储服务（ iSCSI ）。如果基础架构允许，您可以选择启用 FCP 服务。

      • 如果要为磁盘启用异步灾难恢复 DP ，则为灾难恢复区域，分区和计划。有关详细信息，请参见 "灾难恢复"。

    • 确定或定义要映射到磁盘的主机组。您也可以在创建磁盘时创建主机组。

    • 要为磁盘启用异步灾难恢复 DP 选项，必须在启用了异步灾难恢复的块存储上创建磁盘。有关详细信息，请参见 "灾难恢复"。

    • 要为磁盘启用同步灾难恢复，请在启用了 MetroCluster 的区域中创建磁盘。

    • 您可以定义备份策略以按计划捕获磁盘备份。有关详细信息，请参见 "备份"。

    步骤

    1. 查看 "磁盘列表"。

    2. 单击 * 创建磁盘 * 。

    3. 在创建磁盘对话框中，选择创建新磁盘。

      此时将显示创建磁盘页面。

    4. 选择用于访问磁盘的协议： iSCSI 或 FCP 。

    5. 为新磁盘选择子租户，区域和区域以及子租户。此时将显示选定区域，区域和子租户的块存储详细信息。

    6. 如果未在选定子租户的选定区域中启用块存储，则会显示一条消息以对其进行初始化。

    7. 单击 * 初始化块存储 * 。

    8. 在初始化块存储对话框中：

      1. 在 iSCSI 面板中，指定子网。默认情况下，所有磁盘都启用 iSCSI 服务。

      2. 如果需要，请启用 FCP 协议（仅当底层基础架构支持该协议时才可用）。

      3. 如果需要，请启用异步灾难恢复选项（启用并选择区域，区域和计划）。

      4. 如果选定区域已启用 MCC ，则同步灾难恢复切换将处于启用状态，并且无法禁用。在此块存储中创建的磁盘会同步复制到同步灾难恢复面板中显示的分区。

      5. 单击 * 创建 * 以初始化块存储。请等待，直到块存储初始化为止。此时将返回创建磁盘页面。

    9. 填写以下字段：

      字段 Description

      Name

      输入磁盘名称。

      磁盘路径

      输入磁盘的路径。

      操作系统类型

      为磁盘选择操作系统。

      主机组

      此列表显示与为磁盘选择的协议，操作系统类型，子租户和分区匹配的现有主机组。选择一个或多个主机组。要定义新的主机组，请参见下一步。

    10. 如果需要，请创建新主机组：

      1. 单击 * 创建主机组 * 。此时将显示创建主机组对话框。

      2. 指定主机组的名称。

      3. 为组添加启动程序。对于每个启动程序，指定别名和启动程序。

      4. 如果需要，请在标记部分的主机组中添加标记（键值对）。

      5. 单击 * 创建 * 。系统将创建主机组，并在成功创建主机组后显示一条消息。

      6. 要将新创建的主机组映射到磁盘，请转到主机组字段并将其选中。

    11. 选择性能服务级别。

      选择一个选项以查看该级别的性能详细信息（如峰值 / 预期 IOPS/throughput ）。选择最符合您需求的服务级别。

    12. 指定磁盘的容量。

    13. 如果此磁盘需要 Snapshot ：

      1. 切换以启用 Snapshot 策略以查看 Snapshot 策略字段。

      2. 指定何时创建快照：

        • * 每小时。 * 指定创建快照的分钟（每小时）（选中）以及要保留的每小时快照数。

        • * 每日。 * 指定创建快照的时间（小时和分钟）（选中）以及要保留的每小时快照数。如果要指定每天必须创建 Snapshot 的多个小时，则可以将小时值添加到逗号分隔列表中，例如 5 ， 10 ， 15 ， 等等。

        • * 每周。 * 指定创建快照的时间（星期几，小时和分钟）（选中）以及要保留的每周快照数。

        • * 每月。 * 指定创建 Snapshot 的时间（日期，小时和分钟）以及要保留的每月快照数。

    14. 如果在底层块存储上启用了异步灾难恢复，则会自动为新磁盘启用异步灾难恢复复制。如果要从异步灾难恢复复制中排除磁盘，请切换异步灾难恢复切换以禁用异步灾难恢复。

    15. 如果要在已启用 MetroCluster 的区域中创建磁盘，则同步灾难恢复按钮将处于启用状态，并且无法禁用。磁盘将复制到同步灾难恢复面板中显示的区域。

    16. 为此磁盘启用备份：

      1. 切换以启用备份策略以查看备份策略字段。

      2. 指定备份区域。

      3. 指定要保留的每种备份类型的数量：每日，每周和 / 或每月。

    17. 如果要向磁盘添加标记（键值对），请在标记部分中指定这些标记。

    18. 单击 * 创建 * 。此操作将创建一个作业来创建磁盘。

    创建磁盘将作为异步作业运行。您可以

    • 在作业列表中检查作业的状态。

    • 作业完成后，在磁盘列表中检查磁盘的状态。
