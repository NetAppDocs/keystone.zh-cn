= 查看用户列表
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
用户列表显示登录用户有权访问的租约中的所有用户的列表。要查看此列表，请从菜单中选择管理 > 用户。

此列表显示有关每个用户的简单信息，例如用户名，名字和姓氏，电子邮件地址，主租户和创建日期。

通过每个用户旁边的操作图标，您可以修改或删除该用户。
