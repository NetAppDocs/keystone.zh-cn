= 查看存储分段
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
" 分段 " 列表显示选定子租户的现有分段。

. 要查看此列表、请从菜单中选择*对象存储>分段*。
. 如果需要，可使用子租户下拉菜单选择其他子租户。
+
显示每个存储分段的简单信息，例如：

+
** Name
** Region
** Size
** 创建日期
** 对象
** 操作(例如删除存储分段)
