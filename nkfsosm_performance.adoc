= 性能服务级别
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
NetApp Keystone 灵活订阅（ Flex 订阅）可按预定义的性能级别提供容量。

每个性能服务级别均通过其 I/O 密度来定义，该密度是指性能（每秒输入 / 输出操作数 IOPS ）与已用存储（已存储数据的 TiB ）（即 IOPS/TiB ）之比。

由 Flex 订阅服务管理的每个卷都与一个性能服务级别相关联。在计算卷的 I/O 密度时会考虑相应卷上使用的所有 I/O 操作和所有已用存储。
