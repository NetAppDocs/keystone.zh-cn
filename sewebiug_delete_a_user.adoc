= 删除用户
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
使用此方法删除用户。


NOTE: 无法恢复已删除的用户。

.步骤
. 从菜单中选择管理 > 用户。
. 在列表中找到该用户，然后单击删除图标。
. 在确认删除对话框中，输入要删除的用户的名称。
. 单击 * 确认 * 。
