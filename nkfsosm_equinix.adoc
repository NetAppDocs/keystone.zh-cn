= 由Equinix提供支持的Keystone服务
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
NetApp与Equinix合作、在Equinix数据中心托管NetApp Keystone 服务、以确保为您提供统一的解决方案。

Equinix支持的Keystone服务与标准Keystone服务产品保持不变。

除了标准Keystone服务产品之外、您还需要选择一个Equinix数据中心来托管Keystone设备。
